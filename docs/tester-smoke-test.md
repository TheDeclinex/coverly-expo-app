# Coverly Tester Smoke Test

Use this checklist for trusted Android APK and iOS TestFlight builds. Keep notes short and attach screenshots where possible.

## Install

### Android preview APK

1. Open the APK link on the Android test phone.
2. Allow installation from the browser or file manager if prompted.
3. Install Coverly.
4. Open the app and confirm the version/build shown on the Account screen.

### iOS TestFlight

1. Install TestFlight from the App Store.
2. Open the Coverly TestFlight invite.
3. Install the latest available build.
4. Open the app and confirm the version/build shown on the Account screen.

## Smoke Test Path

1. Sign up with a tester email, or sign in with an existing tester account.
2. Create a property with a contents cover amount.
3. Add at least one room to the property.
4. Run a single photo scan and confirm detected items appear in review.
5. Run a multi-photo scan with at least two photos and confirm the review thumbnails make sense.
6. Run a short video scan and confirm frame preparation and scan review complete.
7. Save one detected item from scan review.
8. Save all remaining detected items from scan review.
9. Open an item detail page.
10. Edit item name, quantity, and price from the item edit modal.
11. Expand Product & Purchase Details, add a value, save, and revisit the item.
12. Check item photos open full-screen and pins appear where expected.
13. Run replacement pricing on one item.
14. Add evidence to an item, such as a receipt image or PDF.
15. Open Claim Packs and start a draft for a property.
16. Visit Account, Profile & preferences, Feedback & support, Privacy, and Terms.
17. Send one feedback report, with a screenshot if relevant.

## What To Report

For every issue, include:

- Phone model.
- Android or iOS version.
- Coverly build link, version, and build number.
- Screenshot or screen recording if possible.
- The exact action immediately before the error.
- Whether retrying fixed it.
- Whether the issue happened on Wi-Fi, mobile data, or both.

## High-Priority Issues

Report these immediately:

- App cannot install or open.
- Login or signup cannot complete.
- Property or room creation fails.
- Any scan mode cannot reach review.
- Scan review cannot save items.
- Photos stay as grey placeholders after leaving and returning.
- Item edit save/cancel requires two actions.
- Replacement pricing cannot start or save.
- Evidence upload fails repeatedly.
- Feedback/support cannot send.
