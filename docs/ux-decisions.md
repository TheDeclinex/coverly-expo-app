# UX and Brand Decisions

## Brand direction

Coverly should feel:
- Calm.
- Trustworthy.
- Modern.
- Useful.
- Insurance-aware without feeling corporate or intimidating.

Preferred palette direction:
- Teal.
- Slate.
- Soft white / off-white.
- Muted neutrals.

Avoid:
- Too much bright green.
- Heavy dark green everywhere.
- Overly playful visuals.
- Empty screens with no guidance.

## Key product language

Tagline:

> Know what you own.

Preferred plain-language positioning:
- Home contents inventory.
- Replacement value.
- Insurance-ready evidence.
- Claim-ready pack.
- Value vs insured amount.

## Primary actions

Scanning is the primary path.

Use:
- “Scan items”
- “Scan room”
- “Add photos”
- “Add item manually”

Manual entry should be available but secondary.

## Empty-state guidance

New users should not land in a blank app.

Good empty-state guidance:
- Create your first property.
- Add your insured contents value.
- Add rooms.
- Scan a room.
- Review detected items.
- Add replacement prices.
- Build evidence over time.

## Onboarding

Current onboarding direction:
1. Welcome / value proposition.
2. Explain AI scanning, room organisation, replacement pricing.
3. Create first property.
4. Land user in a guided first-use state.

Recommended improvement:
- Ask for property name.
- Ask for insured contents value.
- Then guide the user to add rooms / scan first room.

## Completion ring

Completion rings are used to encourage users to fully document rooms.

Intent:
- Show progress toward a room being documented.
- Make progress feel rewarding.
- Avoid visual clutter.

Rules:
- Background ring should be subtle slate/grey.
- Completion colour can use teal or a soft gradient.
- Ring should animate gently on load.
- At 100%, show a small, tasteful congratulations moment.
- Avoid making every card feel overly green.

## Pins

Preferred direction:
- Clean pin-drop style.
- Centered inner dot/ring.
- Teal/white variants.
- No unnecessary bulky outer ring.
- Must remain visible over photos.

## Splash/login

Splash/login should feel more modern and tech-friendly.

Should include:
- Coverly brand.
- Tagline.
- Clear sign in/sign up path.
- Trust/security reassurance such as “Your data is encrypted and secure.”

Avoid:
- Too plain.
- Too dark.
- Too green.
