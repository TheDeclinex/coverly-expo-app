Use this exact schema:

Top-level property/file table:

* table: `inventory_files`
* primary key: `id` text
* owner column: `user_id` uuid
* display fields:

  * `name`
  * `status`
  * `contents_sum_insured`
  * `created_by_email`
  * `property_type`
  * `property_cover_image_url`
  * `created_date`
  * `last_modified`

Rooms table:

* table: `inventory_rooms`
* primary key: `id` uuid
* property/file link: `file_id` text → `inventory_files.id`
* owner column: `user_id` uuid
* display fields:

  * `name`
  * `room_type`
  * `sort_order`
  * `cover_photo_url`
  * `notes`
  * `description`
  * `archived_at`

Items table:

* table: `inventory_items`
* primary key: `id` text
* property/file link: `file_id` text → `inventory_files.id`
* room link: `room_id` uuid → `inventory_rooms.id`
* display fields:

  * `name`
  * `category`
  * `confidence`
  * `estimated_price`
  * `unit_estimated_price`
  * `quantity`
  * `quantity_estimate`
  * `valuation_basis`
  * `description`
  * `image_url`
  * `photo_url`
  * `notes`
  * `brand_maker`
  * `model_series`
  * `condition_label`
  * `purchase_source`
  * `original_purchase_price`
  * `purchase_year_approx`
  * `web_listing_url`
  * `web_listing_title`
  * `web_listing_price`
  * `web_listing_source`
  * `sort_order`

Query rules:

* Properties screen: select from `inventory_files`; RLS should restrict results to the logged-in user.
* Rooms screen: select from `inventory_rooms` where `file_id = selected inventory_files.id` and `archived_at is null`, ordered by `sort_order`.
* Items screen: select from `inventory_items` where `room_id = selected inventory_rooms.id`, ordered by `sort_order`.
* Item detail screen: select from `inventory_items` where `id = selected item id`.
